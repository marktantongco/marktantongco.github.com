7-Day Quick-Start

Own Your Results. Print this. Check boxes. Win.

Day 1 Mindset Reset
  • Read Pillars 1-3 (Playbook pages 3-6)
  • Rewrite 3 blame-language sentences from your notes
  • 60-second pre-call prayer/centering practice
  • Write your personal commitment statement for this week
Day 2 Script Practice
  • Record yourself using the Ownership Script (voice memo)
  • Listen back. Adjust tone to calm + curious
  • Use script with 1 real client/prospect today
  • Journal: How did it feel to use ownership language?
Day 3 System Setup
  • Import Accountability Tracker into Google Sheets
  • Set Friday 4:30 PM recurring calendar event: "Weekly Reset"
  • Save Client Follow-Up System to Notion/Docs
  • Set up your CRM pipeline tags (Warm/Hot/Closed/Cold)
Day 4 Team Alignment
  • Share Huddle Agenda template with your team
  • Run first 15-min huddle using the script
  • Ask for 1 piece of feedback on the format
  • Skip this if you're a solo advisor — use the time for client outreach
Day 5 Client Application
  • Identify 3 warm leads using the 3-Touch Rule
  • Send Touch 1 to all 3
  • Log interactions in your tracker
  • Before each outreach: 10-second service prayer
Day 6 Reflection
  • Complete Week 1 row in Accountability Tracker
  • Journal: "Where did I feel resistance to ownership? What's beneath that?"
  • Celebrate 1 win (seriously — do it)
  • Share your #1 takeaway: marktantongco.typeform.com/to/playbook-takeaway
Day 7 Plan Forward
  • Review Week 1 reflection
  • Set 3 priorities for Week 2
  • Share one insight with a peer (accountability multiplies)
